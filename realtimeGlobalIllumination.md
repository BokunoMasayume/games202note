# 实时全局光照 （screen space）

- screen space ambient occlusion (ssao)
  - 全局光照的近似
  - screen space
  - key idea
    - 我们不知道简洁光照
    - 假设对每个着色点， 每个方向全局光照相同
    - 考虑每个着色点对各个方向的vibility有不同
    - 假设diffuse材质
